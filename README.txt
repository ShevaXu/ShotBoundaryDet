The implementation of a shot boundaries detection method described in 
"Yuan et. al. A formal study of shot boundary detection. IEEE TRANSACTIONS ON CIRCUITS AND SYSTEMS FOR VIDEO TECHNOLOGY, 2007."
(http://ieeexplore.ieee.org/xpls/abs_all.jsp?arnumber=4079667)

Add handles of over-cast graphics shot transistion for soccer broadcast video specifically.

Shot boundary detection data can be acquired at the TRECVID website (http://trecvid.nist.gov/trecvid.data.html).

--usage
change the video file url in param.xml, the run main.

Project started in 2010, cleaned up on 2014/4/22.
